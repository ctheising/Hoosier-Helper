    Introduction

    This is a basic technical documentation of essential methods and functions you might need for your own integrations, customization or references.

    When you make customizations, please do take in consideration future updates (especially breaking changes versions), by not modifying directly the essential core files (jquery.calendar.js | calendar.php | calendar.extension.php). Take advantage of the public methods to create your own files.

    In case you have modified your files, check the changelog file for the modified files so you quickly apply the changes to your modification but its advised that you remain the core functions, variables and naming.

    For fullcalendar core functions, refer to this documentation: https://fullcalendar.io/docs/v3
